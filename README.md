Core Python library for [foss42 Open Source APIs](https://github.com/foss42/api). 

## Installation

```
pip3 install foss42
```

## Testing

Install `pytest`

```
$ pip3 install pytest
```

Clone the repo, set it as the current working directory and run the tests using the command:

```
$ pytest
```
